# Salve Mundi website API

## Description

This is a NodeJS backend build with TypeORM and NestJS to deliver all of the data to the website. 

## Getting started

```bash
$ npm prune
$ npm install
$ npm run start:dev
```

## Run tests

```bash
# run tests with coverage report
$ npm run test:cov
```
